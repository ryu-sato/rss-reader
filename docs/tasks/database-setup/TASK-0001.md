## 実装詳細

### schema.prisma

```prisma
datasource db {
  provider = "sqlite"
  url      = env("DATABASE_URL")
}

generator client {
  provider        = "prisma-client-js"
  output          = "../src/generated/prisma"
}

model Feed {
  id            String    @id @default(uuid())
  url           String    @unique
  title         String
  description   String?
  memo          String?
  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt
  lastFetchedAt DateTime?
}
```

### Feedモデルフィールド

| フィールド | 型 | 制約 | 説明 |
|----------|---|------|------|
| id | String | PK, UUID | 主キー |
| url | String | UNIQUE | フィードURL |
| title | String | NOT NULL | フィードタイトル |
| description | String? | nullable | フィード説明 |
| memo | String? | nullable | ユーザーメモ |
| createdAt | DateTime | auto | 作成日時 |
| updatedAt | DateTime | auto | 更新日時 |
| lastFetchedAt | DateTime? | nullable | 最終取得日時 |
