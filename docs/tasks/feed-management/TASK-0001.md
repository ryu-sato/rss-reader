## 実装詳細

RSSフィード管理のビジネスロジック層。CRUD操作を担う5つの関数を実装。

### createFeed(url: string) → Feed
1. URLが既に登録済みか重複チェック（ConflictError: 409）
2. SSRF Guardによるセキュリティ検証
3. rss-fetcherでRSSメタデータ（title, description）取得
4. Prisma経由でDBにレコード作成
5. 作成したFeedオブジェクトを返却

### getAllFeeds() → FeedListItem[]
- 全フィードをcreatedAt降順で取得
- FeedListItem型（id, url, title, description, createdAt）のみ返却

### getFeedById(id: string) → Feed
- IDでフィードを検索
- 未存在時: NotFoundError (404)

### updateFeed(id: string, data: UpdateFeedRequest) → Feed
- title, description, memoフィールドを更新
- 更新前に対象フィードの存在を確認

### deleteFeed(id: string) → void
- IDでフィードを削除
- 削除前に対象フィードの存在を確認
