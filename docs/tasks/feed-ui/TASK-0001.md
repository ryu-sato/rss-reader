## 実装詳細

Next.js App Routerのルートレイアウトとグローバルスタイル定義。

### layout.tsx
- `<html lang="ja">` 設定（日本語）
- Geistフォントファミリー適用
- スティッキーヘッダー: RSSアイコン（lucide-react）＋「RSS Reader」タイトル
- グローバルコンテナスタイリング
- SonnerトーストプロバイダーをThemeProvider内に配置

### globals.css
- Tailwind CSS v4 (`@import "tailwindcss"`) 使用
- CSSカスタムプロパティによるデザイントークン:
  - `--background`, `--foreground` (ライト/ダーク)
  - `--primary`, `--secondary`, `--muted`, `--destructive`, `--accent`
  - `--border`, `--input`, `--ring`
  - `--radius` (ボーダー半径トークン)
  - `--chart-1` ～ `--chart-5` (チャート用カラー)
- OKLch色空間を使用した色定義
- `@layer base` でグローバルにborderとoutline色を適用
