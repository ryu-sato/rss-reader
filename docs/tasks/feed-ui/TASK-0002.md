## 実装詳細

### page.tsx (サーバーコンポーネント)
- `GET /api/feeds` を直接 `feed-service.getAllFeeds()` 経由で呼び出し
- フィード数カウント表示 (`{n}件のフィード`)
- 「フィードを追加」ボタン（/feeds/newへのリンク）
- `FeedList` コンポーネントにフィードデータを渡す

### FeedList (クライアントコンポーネント)
- 各フィードをカード形式でリスト表示
  - フィードアイコン（lucide-react: Rss）
  - タイトル
  - URL（グレーテキスト）
  - 作成日時
- 編集ボタン → `/feeds/[id]/edit` へのリンク
- 削除ボタン → `DeleteConfirmDialog` を開く
- 空状態表示: 「まだフィードが登録されていません」+ 「最初のフィードを追加する」CTA
