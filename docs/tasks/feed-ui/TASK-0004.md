## 実装詳細

### page.tsx (サーバーコンポーネント、動的ルート)
- `params.id` でフィードIDを取得
- `feed-service.getFeedById(id)` でフィードデータ取得
- 未存在時: `notFound()` で404ページ表示
- `EditFeedForm` コンポーネントにフィードデータを渡す

### EditFeedForm (クライアントコンポーネント)
- フィードURLを読み取り専用ボックスで表示（変更不可）
- React Hook Form + Zod バリデーション
- 編集可能フィールド:
  - title（必須）
  - description（任意、最大1000文字、文字数カウンター付き）
  - memo（任意、最大1000文字、文字数カウンター付き）
- 初期値: 既存フィードデータをフォームにセット
- 送信処理:
  - `PUT /api/feeds/[id]` へリクエスト
  - 送信中のローディング状態（ボタン無効化）
  - 成功後ホームへリダイレクト
- キャンセルボタン: ホームへナビゲート
