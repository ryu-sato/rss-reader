## 実装詳細

フィード削除前のユーザー確認を行うダイアログコンポーネント。

### Props
```typescript
interface DeleteConfirmDialogProps {
  feedId: string
  feedTitle: string
}
```

### 動作フロー
1. 赤色（destructive）の「削除」ボタンを表示
2. クリックでAlertDialogを開く
3. ダイアログ内にフィードタイトルを表示してユーザーに確認
4. 「キャンセル」ボタン: ダイアログを閉じる
5. 「削除する」ボタン（赤色）:
   - `DELETE /api/feeds/[feedId]` を呼び出し
   - 成功時: `router.refresh()` でページをリフレッシュ
   - エラー時: エラーメッセージを表示
