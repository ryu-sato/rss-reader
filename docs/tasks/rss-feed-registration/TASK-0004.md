## タスク概要

`lib/ssrf-guard.ts` を実装する。入力URLのホスト名をDNS解決し、解決先IPアドレスがプライベートIP（RFC1918）・ループバック・リンクローカル等に該当する場合はリクエストをブロックする。
また、URL形式（http/https、2048文字以内）の基本バリデーションも担当する。

## 実装詳細

### 1. SSRFガードの実装 🔵

**信頼性**: 🔵 *architecture.md NFR-101・design-interview.md Q5より*

**実装ファイル**: `lib/ssrf-guard.ts`

```typescript
import dns from 'dns/promises'
import { SSRFError } from './errors'

// プライベートIPアドレスのCIDRリスト（RFC1918 + 特殊アドレス）
const PRIVATE_IP_RANGES = [
  // IPv4
  /^127\./,           // ループバック
  /^10\./,            // RFC1918 Class A
  /^172\.(1[6-9]|2[0-9]|3[01])\./,  // RFC1918 Class B
  /^192\.168\./,      // RFC1918 Class C
  /^169\.254\./,      // リンクローカル
  /^100\.(6[4-9]|[7-9][0-9]|1[01][0-9]|12[0-7])\./,  // CGNAT
  /^0\./,             // 未指定
  // IPv6
  /^::1$/,            // ループバック
  /^fe80:/i,          // リンクローカル
  /^fc00:/i,          // ユニークローカル
  /^fd/i,             // ユニークローカル
]

function isPrivateIP(ip: string): boolean {
  return PRIVATE_IP_RANGES.some(range => range.test(ip))
}

export async function validateUrl(url: string): Promise<void> {
  // URL形式チェック
  let parsedUrl: URL
  try {
    parsedUrl = new URL(url)
  } catch {
    throw new SSRFError('Invalid URL format')
  }

  // スキームチェック
  if (!['http:', 'https:'].includes(parsedUrl.protocol)) {
    throw new SSRFError('URL must start with http:// or https://')
  }

  // URL長チェック
  if (url.length > 2048) {
    throw new SSRFError('URL must be 2048 characters or less')
  }

  // DNS解決してIPアドレスを取得
  try {
    const addresses = await dns.lookup(parsedUrl.hostname, { all: true })
    for (const addr of addresses) {
      if (isPrivateIP(addr.address)) {
        throw new SSRFError('URL is not allowed')
      }
    }
  } catch (err) {
    if (err instanceof SSRFError) throw err
    throw new SSRFError('URL is not allowed')
  }
}
```
