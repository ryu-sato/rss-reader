## 実装詳細

### 1. フィード編集ページの実装 🔵

**信頼性**: 🔵 *architecture.md ディレクトリ構造・user-stories.md 3.1・api-endpoints.md PUT /api/feeds/:idより*

**実装ファイル**: `app/feeds/[id]/edit/page.tsx`

```typescript
import { getFeedById } from '@/lib/feed-service'
import EditFeedForm from '@/components/edit-feed-form'
import Link from 'next/link'
import { notFound } from 'next/navigation'
import { ArrowLeft } from 'lucide-react'

interface Props {
  params: { id: string }
}

export default async function EditFeedPage({ params }: Props) {
  let feed
  try {
    feed = await getFeedById(params.id)
  } catch {
    notFound()
  }

  return (
    <main className="container mx-auto py-8 px-4 max-w-2xl">
      <Link href="/" className="flex items-center gap-2 text-muted-foreground hover:text-foreground mb-6">
        <ArrowLeft className="h-4 w-4" />
        Back to feeds
      </Link>
      <h1 className="text-2xl font-bold mb-6">Edit Feed</h1>
      <EditFeedForm feed={feed} />
    </main>
  )
}
```

### 2. EditFeedFormコンポーネントの実装 🔵

**信頼性**: 🔵 *REQ-104・api-endpoints.md PUT /api/feeds/:idより*

**実装ファイル**: `components/edit-feed-form.tsx`

- URL表示（読み取り専用）
- タイトル入力（必須）
- 説明入力（Textarea、1000文字以内）
- メモ入力（Textarea、1000文字以内）
- 「Save Changes」ボタン（送信中はローディング表示）
- 「Cancel」リンク（`/` へ）

送信処理:
```typescript
const response = await fetch(`/api/feeds/${feed.id}`, {
  method: 'PUT',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ title, description, memo }),
})
// 成功後 router.push('/')
```
