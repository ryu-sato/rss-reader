## 実装詳細

URLを受け取り、RSSフィードのメタデータ（title, description）を取得して返す関数。

### fetchRssFeed(url: string) → { title, description, lastFetchedAt }

1. `fetch(url, { signal: AbortSignal.timeout(30000) })` でHTTPリクエスト
2. レスポンスが非OKの場合 `FeedFetchError` をスロー
3. `rss-parser` でRSS/Atom解析
4. フォーマット不正時 `InvalidFeedFormatError` をスロー
5. title が空の場合 url をフォールバックとして使用
6. `lastFetchedAt: new Date()` を付加して返却

### 対応フォーマット
- RSS 2.0
- Atom 1.0

### エラーケース
- ネットワークエラー → `FeedFetchError`
- タイムアウト（30秒）→ `FeedFetchError`
- 非200レスポンス → `FeedFetchError`
- 無効なRSSフォーマット → `InvalidFeedFormatError`
